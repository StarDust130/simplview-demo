# Engineering Rules

Use Next.js App Router.

Use TypeScript.

Use Tailwind CSS.

Use shadcn/ui.

Use Framer Motion.

Use Lucide icons.

Create reusable components.

Create reusable hooks.

No duplicated code.

No inline styles.

No hardcoded colors.

No large components.

Follow clean architecture.

Follow modern React patterns.

Accessibility matters.

Performance matters.

Everything should be production ready.
