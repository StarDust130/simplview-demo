# UI Rules

Large whitespace.

Excellent typography.

Rounded corners.

Soft shadows.

Smooth transitions.

Perfect alignment.

Consistent spacing.

Simple icons.

Beautiful cards.

Readable text.

Large headings.

Professional hierarchy.

Everything must align perfectly.

No visual clutter.

No unnecessary borders.

No inconsistent spacing.

Everything should feel expensive.

If something can be simplified,
simplify it.

Always prioritize clarity.
